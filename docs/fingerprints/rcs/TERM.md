## TERM -- `0x5445524D`

```
C   ( -- )      Clear the screen
D   ( n -- )    Move cursor down n lines
G   (x y -- )   Put cursor at position x,y (home is 0,0)
H   ( -- )      Move cursor to home
L   ( -- )      Clear to end of line
S   ( -- )      Clear to end of screen
U   ( n -- )    Move cursor up n lines
```
